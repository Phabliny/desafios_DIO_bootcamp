O que é HTML?

O HTMLé o componente básico da web, ele permite inserir o conteúdo e estabelecer a estrutura básica de um website.

Segue um trecho do site wikipedia:

Documentos HTML podem ser interpretados por navegadores. Todo documento HTML possui marcadores (inglês: tags), palavras entre parênteses angulares (chevron) (< e >); esses marcadores são os comandos de formatação da linguagem. Um elemento é formado por um nome de marcador (tag), atributos, valores e filhos (que podem ser outros elementos ou texto). Os atributos modificam os resultados padrões dos elementos e os valores caracterizam essa mudança.

A tag font era utilizadano HTML 4 para especificar a face da fonte, o tamanho da fonte e a cor do texto. Entretanto não é utilizada no HTML5, sendo substituida pela tag style. Esse é um exemplo de texto com a cor e fonte modificados pela tag style.
